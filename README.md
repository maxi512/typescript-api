# typescript-api
Setting up a repo with a simple API made with Typescript

Clone this repo:
```shell
git clone https://github.com/maxi512/typescript-api.git
```
Install dependencies:
```shell
npm install
```
Build:
```shell
npm run build
```
Run it:
```shell
npm start
```

Run it with docker-compose:
```shell
docker-compose up
```
